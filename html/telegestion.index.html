Télégestion d'une serre

Lycée Louis Pasteur, Hénin-Beaumont, Mars 2019 - Juin 2019
Projet du baccalauréat

Lors de mon baccalauréat, nous avons eu par groupe de 4 de faire l’automatisation ainsi que la gestion d’une serre à distance via son ordinateur ou smartphone.

Le projet se composait en 4 parties :
- Gestion de l’arrosage
- Gestion des ouvrants
- Gestion du chauffage
- Gestion de l’éclairage

J’avais comme objectif de faire la gestion de l’éclairage via un capteur de lumière pour gérer l’automatisation.

Pour la réalisation nous avons utilisé ici le framework Arduino.
